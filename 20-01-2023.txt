	2] To view particular command details then use bellow command
			syntax: 	git help <commandname>
			
			example: 	git help init
			
		Once we enter above command one browser will open and display information about that command.

	 Untracked file:
		The files which are created newly in our git folder are called untracked files.
		Untracked files are displaying in red colour.
	 git status
		above command will display untracked and staged files.
		
	 git add Demo.java
		above command will moving file Demo.java from untracked area to staged area.
		staged files are displaying green colour.
	 git status
